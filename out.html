md2html – Markdown → HTML 変換CLIツール

Goで作成した、MarkdownファイルをHTMLに変換するシンプルなCLIツールです。
ローカル環境でもDocker環境でも動作します。

特徴

  • Markdownファイル（.md）をHTMLに変換
  • 出力ファイル名・タイトルを指定可能
  • 完全HTML（--standalone）形式に対応
  • ローカル実行・Docker実行の両方に対応

インストール方法

1. ローカル実行（Goインストール済みの場合）

git clone https://github.com/yourname/md2html.git
cd md2html
go build -o md2html

2. Dockerを使う場合

git clone https://github.com/yourname/md2html.git
cd md2html
docker build -t md2html:latest .

使い方

基本構文

md2html <入力Markdown> [オプション]

主なオプション

オプション 説明 例
-o, --output 出力HTMLファイル名 -o output.html
--title HTMLタイトル設定 --title "My Doc"
--standalone 完全HTMLとして出力 --standalone

実行例（ローカル環境）

md2html README.md

→ README.html が生成されます。

md2html README.md -o out.html --title "ドキュメント"

→ タイトルを変更して out.html を生成します。

md2html README.md -o out.html --standalone

→ <html> から始まる完全HTMLを生成します。

実行例（Docker環境）

カレントディレクトリの README.md を変換して out.html を作成：

docker run --rm -v "$(pwd)":/work -w /work md2html:latest README.md -o out.html

動作確認用コマンド（Windows cmd.exe）

基本

md2html README.md

出力指定

md2html README.md -o out.html

タイトル指定

md2html README.md -o out.html --title "My Document"

完全HTML

md2html README.md -o out.html --standalone --title "Standalone Doc"

テスト方法

  1. サンプルの README.md を用意
  2. 上記のコマンドを順番に実行
  3. 出力HTMLをブラウザで開いて表示を確認
